# Credit_Risk_Prediction
# Credit Risk Prediction using Machine Learning

## Objective
To predict whether an individual is creditworthy based on historical financial data.

## Dataset
The dataset contains financial attributes such as age, income, credit amount, loan duration, employment status, savings, and housing information.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Evaluation Metrics
- Precision
- Recall
- F1-Score
- ROC-AUC

## Project Workflow
- Data Cleaning & Feature Engineering
- Encoding & Scaling
- Model Training and Comparison
- Performance Evaluation
- Final Model Selection

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Result
Random Forest performed best and was selected as the final model.
